# 사용 가이드

## 👤 일반 사용자

### 교원 현황 조회

1. [https://parkseihuan.github.io/Faculty-status](https://parkseihuan.github.io/Faculty-status) 접속
2. 자동으로 최신 교원 현황 데이터가 로드됩니다
3. 교원명을 클릭하면 상세 정보를 확인할 수 있습니다

### 주요 기능

- **🔄 데이터 새로고침**: 최신 데이터로 업데이트
- **🖨️ 인쇄**: 현재 페이지를 인쇄용으로 출력
- **교원 상세 정보**: 교원명을 클릭하여 생년월일, 임용일, 재직 상태 등 확인

### 데이터 표시 방식

#### 색상 구분

- **배경색**:
  - 🔴 빨강 (긴급): 임용 종료일 1-3개월
  - 🟠 주황 (주의): 임용 종료일 4-6개월
  - 🟡 노랑 (관심): 임용 종료일 7-9개월
  - 🟢 연두 (양호): 임용 종료일 10-12개월
  - 💙 연파랑: 12개월 이내 정년 도래 예정

- **텍스트 색상**:
  - 🔵 파란색: 휴직 중
  - 🟢 초록색: 연구년

#### 기타 표시

- **진한 글씨**: 연봉제 전임교원
- **숫자 표시**: 재임용/정년까지 남은 개월 수 (12개월 이하만 표시)

## 🔐 관리자

### 로그인

1. 메인 페이지에서 "🔐 관리자" 버튼 클릭
2. 관리자 비밀번호 입력
3. 로그인 성공 시 관리자 페이지로 이동

### 엑셀 파일 업로드

#### 1. 엑셀 파일 준비

파일 형식: `교원현황_YYYYMMDD_HHMMSS.xlsx`

**필수 컬럼:**
- 대학
- 소속
- 직번
- 성명
- 직렬
- 직급
- 성별
- 재직구분
- 호봉
- 최초임용 시작일
- 최초임용 종료일
- 재임용종료일
- 생년월일
- 정년일자

#### 2. 업로드 방법

**방법 1: 드래그 앤 드롭**
- 파일을 업로드 영역으로 드래그

**방법 2: 파일 선택**
- "파일 선택" 버튼 클릭
- 파일 선택 후 "업로드" 버튼 클릭

#### 3. 업로드 확인

- 업로드가 성공하면 처리된 인원 수가 표시됩니다
- 실패 시 에러 메시지 확인 후 재시도

### 조직 순서 설정

#### 1. 현재 설정 불러오기

- "현재 설정 불러오기" 버튼 클릭
- 현재 조직 구조가 표시됩니다

#### 2. 조직 편집

**대학 편집:**
- 대학명을 직접 수정 가능
- ▲/▼ 버튼으로 순서 변경
- "삭제" 버튼으로 대학 제거
- "+ 대학 추가" 버튼으로 새 대학 추가

**학과 편집:**
- 학과명을 직접 수정 가능
- "삭제" 버튼으로 학과 제거
- "+ 학과 추가" 버튼으로 새 학과 추가

#### 3. 저장

- "저장" 버튼을 클릭하여 변경사항 적용
- 저장 후 일반 사용자 페이지에서도 변경된 순서로 표시됩니다

#### 4. 기본값으로 초기화

- "기본값으로 초기화" 버튼으로 원래 조직 구조로 복원
- 저장을 눌러야 적용됩니다

### 업로드 기록

- 최근 업로드된 파일 목록을 확인할 수 있습니다
- 각 업로드별 처리된 인원 수, 파일 크기 등 정보 표시
- 최대 10개의 기록이 유지됩니다

'겸임조교수' → '겸임교원'
'시간강사' → '강사'
등등...

- **백업**: 업로드 전 기존 엑셀 파일 백업 보관
- **테스트**: 중요한 변경 전 소규모 테스트 파일로 먼저 확인

## 🔒 개인정보 보호

- 업로드된 엑셀 파일은 서버에서 자동 삭제됩니다
- JSON 형태로 변환된 데이터만 서버에 저장됩니다
- 데이터는 암호화된 HTTPS 통신으로 전송됩니다
- 관리자 비밀번호는 bcrypt로 해시화되어 저장됩니다

## ❓ 자주 묻는 질문 (FAQ)

### Q: 데이터가 표시되지 않아요

A: 다음을 확인하세요:
1. 관리자가 최소 1회 이상 엑셀 파일을 업로드했는지
2. 브라우저를 새로고침 (Ctrl+F5 또는 Cmd+Shift+R)
3. 브라우저 콘솔(F12)에서 에러 확인

### Q: 임용 종료일 색상이 표시되지 않아요

A: 엑셀 파일에 다음 컬럼이 있는지 확인:
- 재임용종료일
- 최초임용 종료일
- 정년일자

### Q: 특정 교원이 표시되지 않아요

A: 다음을 확인:
1. 재직구분이 "재직", "연구년", "휴직" 중 하나인지
2. 성명 컬럼이 비어있지 않은지

### Q: 조직 순서를 변경했는데 적용이 안 돼요

A: "저장" 버튼을 클릭했는지 확인하세요. 저장하지 않으면 변경사항이 적용되지 않습니다.
