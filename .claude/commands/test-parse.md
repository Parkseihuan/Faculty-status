# Excel 파일 파싱 테스트

지정된 Excel 파일을 파싱하고 결과를 검증합니다.

## 수행할 작업

1. **파일 형식 확인**
   - .xls 또는 .xlsx 확인
   - 파일 크기 확인

2. **헤더 분석**
   - 첫 번째 행에서 컬럼 헤더 추출
   - 필수 컬럼 존재 확인 (이름, 소속, 직급 등)

3. **데이터 파싱**
   - 각 행 파싱 및 데이터 추출
   - 빈 행, 병합된 셀 처리
   - UTF-8 인코딩 확인

4. **데이터 검증**
   - 필수 필드 존재 확인
   - 데이터 타입 검증
   - 중복 데이터 확인

5. **결과 출력**
   - 파싱된 데이터 샘플 (처음 5개)
   - 총 행 수, 유효한 데이터 수
   - 발견된 오류 목록

6. **JSON 변환**
   - 표준 형식으로 변환
   - 출력 파일 생성 (선택사항)

파싱 중 발견된 문제점과 해결 방법을 제안해주세요.
